# Superminimal Theme 0.1.0

Superminimal is a demo theme, very basic, that I use to demonstrate how to integrate the [Kirki Customizer Toolkit](http://kirki.org/) into a WordPress theme.

## Installation

Note that this theme is not in development. It serves as a demo for the companion article on SitePoint.

1. Clone the repository or extract the archive to your `wp-content/themes` directory
1. Verify that the theme is available in your Themes dashboard
